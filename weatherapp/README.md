

A simple weather app that fetches weather data using the OpenWeatherMap API.

## Features
- Search for weather by city name.
- Display temperature, weather description, and humidity.

## Technologies Used
- HTML
- CSS
- JavaScript

## Setup and Running
1. Clone the project or download the files.
2. Open the folder in VS Code.
3. Replace `YOUR_API_KEY` in `script.js` with your OpenWeatherMap API key.
4. Open `index.html` in a browser to run the app.

## API Key
Get your free API key from [OpenWeatherMap](https://openweathermap.org/api) and replace it in the `script.js` file.
